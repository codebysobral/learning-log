## What I Learned
Today I learned how Git tag-based versioning works and how it 
helps manage both local and remote repositories. I also learned 
the difference between lightweight tags (created with `git 
tag`) and annotated tags (created with `git tag -a -m`), which 
include additional metadata and are commonly used for official 
releases.
